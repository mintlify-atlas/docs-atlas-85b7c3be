## What is LangSmith SDK?

LangSmith SDK is a client library that connects your LLM applications to the LangSmith platform for observability, evaluation, and monitoring. Available for both Python and TypeScript, the SDK provides a lightweight way to trace your application's execution, evaluate model performance, and manage datasets.

Whether you're building with LangChain, using OpenAI directly, or working with any other LLM framework, LangSmith SDK helps you understand what's happening inside your application.

## How it works

LangSmith SDK captures execution traces by wrapping your code with decorators (Python) or wrapper functions (TypeScript). Each trace represents a "run" with inputs, outputs, timing, and metadata.

```python
from langsmith import traceable

@traceable
def my_llm_function(user_input: str) -> str:
    # Your LLM call here
    return response
```

```typescript
import { traceable } from "langsmith/traceable";

const myLlmFunction = traceable(
  async (userInput: string) => {
    // Your LLM call here
    return response;
  },
  { name: "my_llm_function" }
);
```

Traces are automatically sent to LangSmith where you can view them in the web UI, analyze performance, and create datasets for evaluation.

## Quick example

Here's how simple it is to start tracing your OpenAI calls:

<CodeGroup>
```python Python
import openai
from langsmith.wrappers import wrap_openai

# Wrap your OpenAI client
client = wrap_openai(openai.Client())

# Use it exactly as before - tracing happens automatically
response = client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[{"role": "user", "content": "Hello, world!"}]
)
```

```typescript TypeScript
import { OpenAI } from "openai";
import { wrapOpenAI } from "langsmith/wrappers";

// Wrap your OpenAI client
const client = wrapOpenAI(new OpenAI());

// Use it exactly as before - tracing happens automatically
const response = await client.chat.completions.create({
  model: "gpt-4o-mini",
  messages: [{ role: "user", content: "Hello, world!" }],
});
```
</CodeGroup>

That's it! Your traces are now visible in the LangSmith web UI.
